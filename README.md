## How to Use Any Client

```bash
# 1. Clone
git clone https://github.com/spinov001-art/<client-name>.git
cd <client-name>

# 2. Install
pip install requests

# 3. Run
python <module>.py
```

Every client follows the same pattern:
- Single `.py` file with a client class
- `requests` as the only dependency
- Runnable `__main__` demo
- Comprehensive README with examples
